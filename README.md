# xz-leyusportsapp-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以笔记或工具型内容为主，不针对任何特定网站或域名，可作为通用前端页面的存档与展示用途。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放各独立 HTML 页面
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下为各个独立 HTML 文件，文件名与内容主题对应。
- 每个 HTML 文件均为自包含页面，不依赖外部资源（或已内联所需资源）。

## 页面归档说明

- 所有页面均为历史归档或临时记录，内容可能随时间更新或调整。
- 页面功能与样式保持简洁，便于直接打开查看。
- 不保证所有页面在所有浏览器或设备上完全兼容，但尽量遵循基本 Web 标准。

## 维护说明

- 本仓库不定期更新，可能新增、修改或删除页面。
